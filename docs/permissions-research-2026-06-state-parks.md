# State-Park Foraging Landscape — June 2026

A 50-state map of whether personal-use foraging (edible fruits, berries, nuts,
and mushrooms) is permitted in each state's **state-park system**, researched
against the **primary administrative-code regulation / statute** for each state.
Companion to `permissions-research-2026-06.md` (which encoded 11 state systems);
this pass covers all 50 uniformly so the landscape is complete and current.

Status key: **allowed** = an explicit personal-use edible exception in the rule;
**allowed-if-posted** = prohibited by default, but a unit/superintendent may
designate exceptions; **permit-required** = removal only with a permit;
**prohibited** = a general plant-protection rule with no edible exception (the
36-CFR-2.1-style default). Conservative default throughout: a protection rule
with no edible carve-out reads as prohibited. All 50 are **verified** against a
current primary source; the adversarial pass made 2 corrections (Missouri
mushrooms, Idaho) and flagged 1 currency error in an already-encoded rule
(Oregon — see below).

Roughly **half** of states prohibit state-park foraging outright; **18 clearly
allow** personal-use edibles. Mushrooms are permitted in **16** of the 50 systems
and split from the general rule in three states.

## B. Allowed only if posted / designated (4)

Default prohibited; an exception exists but is discretionary and unit-specific —
the app should treat these as **prohibited with a "check posted rules" note**
rather than blanket-allowed.

| State | Mechanism | Citation |
|---|---|---|
| California ✓ | Prohibited unless gathering is specifically authorized and posted for that unit; a few units post berry/mushroom exceptions | 14 CCR 4306 |
| Georgia | Default prohibited; a site manager **may** designate certain fruits/berries for a unit (mushrooms excluded) | Ga. Comp. R. & Regs. r. 391-5-1-.04 |
| Nebraska | Default prohibited (plant matter & products); only posted/permitted exceptions | 163 Neb. Admin. Code ch. 5, § 001.14 |
| Wyoming | Default prohibited; narrow NPS-style exception — **superintendent-designated fruits & berries** for personal consumption only | 024-1 Wyo. Code R. § 1-15(b) |

## Notable findings & decisions

- **Oregon's encoded rule is stale — fix it.** `app.js` currently encodes OR at
  "up to **5 gallons** per person per day" ([app.js:6422](../app.js)), but
  OAR 736-010-0055 was amended (PRD 3-2025, effective **March 12, 2025**) down to
  **1 gallon per person per day**. This is a correction to existing code, called
  out as Phase 0 of the work order.
- **Oklahoma just legalized it (2025).** SB 447 created 61 O.S. § 335, a
  statewide statutory right to forage nuts, edible plants, and fungi on
  state-managed public land for personal consumption — effective Nov 1, 2025.
  Worth a "new this year" note in the UI.
- **Three category splits** need species-aware encoding (the rest are uniform per
  mode): **Missouri** (fruit/berries/seeds/nuts yes, mushrooms **no**),
  **Connecticut** (plants/berries no, mushrooms **yes**), and **Minnesota**
  (already encoded — fruit & mushrooms only, nuts no).
- **Hawaii & Tennessee** allow "renewable natural products" (fruits, berries,
  etc.) but do **not** name mushrooms; conservatively, mushrooms are treated as
  not-covered (excluded) in both.
- **Posted/designated states (CA, GA, NE, WY)** should render as *prohibited with
  a "exceptions only where posted" caveat*, not allowed — overstating a
  discretionary unit-level exception would violate the project's ethic. Wyoming's
  is a true NPS-style superintendent-designation (fruits & berries only).
- **State forests / DNR lands frequently differ from parks** and were captured
  secondarily where found (e.g. NY DEC lands, PA forests, MD forests already
  encoded; many "allowed" states extend the same rule to DNR/forest land —
  Vermont, Wisconsin, Michigan, Minnesota explicitly). Natural-area preserves
  are commonly excluded even in allowed states (WA, IL, MN).

## Encoded-state currency audit (11 states)

All 11 previously-encoded state rules were re-checked against current primary
sources. **10 confirmed accurate.** **1 needs updating: Oregon** (5 gal → 1 gal,
above). California, Colorado, Illinois, Maryland, Michigan, Minnesota, New York,
North Carolina, Pennsylvania, and Washington remain correct as encoded.

## Provenance

50 states researched June 16 2026 via a fan-out workflow — one agent per state
read the governing administrative-code section / statute / official park-rules
page (the Minnesota LCC 2025 50-state foraging memo used only as a lead), then a
second independent agent adversarially re-verified each against the primary
source, biased toward catching overstated permission. All 50 returned
**verified** status at **high** confidence. Corrections: Missouri (mushrooms
downgraded to prohibited), Idaho (downgraded to prohibited). Currency flag:
Oregon limit.

NC, TN, TX, and UT were independently re-researched in a second run (their
first-run agents showed a "ended before completion" status in the run UI, though
their results had in fact been captured). All four **re-confirmed** identically:
NC prohibited, TN allowed (mushrooms excluded — the re-run again flagged a
mushroom "allowed" reading as overstated), TX prohibited (no personal-use
foraging; removal only by director permit), UT prohibited (citation refined to
Utah Admin. Code R651-620-2).
