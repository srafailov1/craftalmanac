# New Material Candidates — June 2026 Permissions Pass

Materials with documented harvest permission somewhere in the encoded rules
that are NOT yet in any species catalog. Presented for owner decision; none
have been added to the app. Each would need safety tags, harvest-ethic labels,
iNaturalist taxon IDs, and seasonality before inclusion.

## Food candidates

| Material | Where permitted | Notes | Source |
|---|---|---|---|
| Red mulberry (*Morus rubra*) | Great Smoky Mountains (1 lb/day) | Native; white mulberry (*M. alba*) is the common invasive twin — good invasive-removal angle | GRSM compendium (verified) |
| Mayapple fruit (*Podophyllum peltatum*) | New River Gorge (~1 gal/day) | Ripe fruit only; all other parts toxic — would need strong safety tags | NERI compendium |
| Chokecherry (*Prunus virginiana*) | Rocky Mountain, Glacier | Fits existing cherries group; pits/leaves toxic like other Prunus | RMNP/Glacier compendiums |
| Rose hips (*Rosa* spp.) | Rocky Mountain | Late-season fruit; also a craft/tea material | RMNP compendium |
| Thimbleberry (*Rubus parviflorus*) | Crater Lake, Yosemite, Glacier | Fits brambles group | compendiums |
| Salmonberry (*Rubus spectabilis*) | Mount Rainier | Pacific Northwest bramble | MORA compendium |
| Cranberry (wild, *Vaccinium* spp.) | Olympic (3.5 gal per 2 weeks) | Coastal bogs | OLYM compendium |
| Red elderberry (*Sambucus racemosa/pubens*) | GRSM list | More toxic raw than common elderberry — cooked only; needs careful tags | GRSM compendium (verified) |
| Tanoak acorns (*Notholithocarpus densiflorus*) | Redwood (10 gal/day!) | Major traditional food; leaching required; sudden oak death zones | REDW compendium |
| Mesquite beans (*Prosopis* spp.) | Death Valley | Desert staple; would extend map coverage into arid regions | DEVA compendium |
| Pine nuts / pinyon (*Pinus edulis/monophylla*) | Death Valley; widespread on BLM/USFS | Personal-use amounts; some forests permit-limited | DEVA compendium, BLM |
| Pomegranate, palm fruit (naturalized) | Death Valley (non-native fruit allowance) | Cultivated-escape category like figs | DEVA compendium |
| Himalayan blackberry (*Rubus armeniacus*) | Yosemite (unlimited — invasive) | Strong invasive-removal story; common across the West Coast | YOSE compendium |
| Ramps (*Allium tricuspidatum/tricoccum*) | **Explicitly NOT allowed at GRSM**; allowed on some USFS/state lands | Include only with strong overharvest warnings if ever added | GRSM compendium (verified) |
