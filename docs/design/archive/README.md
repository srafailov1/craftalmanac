# Design archive

Retired design artifacts kept for historical reference. Not part of the
shipping site.

## `prototype/index.html`

The **owner-approved prototype of record** for the 2026 living-map relaunch
(rev 14, `d4252a7`). It proved the design and the conditions/register systems on
demo data; production (`app.js`, `styles.css`, `index.html` at the repo root)
carries the real data, rules, and attribution.

Retired here at Phase 7 cutover. Throughout the migration the prototype was the
tie-breaker on every visual/interaction decision (see
`docs/design/work-order-redesign.md`, `docs/design/prototype-parity-audit.md`,
and the parity passes in `docs/design/decisions.md`). Design docs that cite
`prototype/index.html` now refer to this archived copy.
